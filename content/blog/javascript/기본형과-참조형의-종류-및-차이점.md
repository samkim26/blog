## DATA TYPES

1. Primitive Type (값을 그대로 할당)
  - Number
  - String
  - Boolean
  - null
  - undefined
  
2. Reference Type (값이 저장된 주소값을 할당(참조))
  - Object 
    - Array
    - Function
    - RegExp


참조형 데이터의 동작은 조금 다르다 - 기본형 데이터와 선언 과정은 같지만 할당 과정이 다르다.
